# 🌐 To-do list Website

Welcome to my To-do list website! This project showcases my work, skills, and experience in a professional format. Built using modern web technologies, it's designed to be responsive, clean, and easy to navigate.

# 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript / React / Vue

- **Backend**: Node.js / Express / Django / Firebase (optional)

- **Database**: MongoDB / Firebase / LocalStorage


# 🚀 Features

- Add new tasks

- Edit existing tasks

- Mark tasks as completed

- Delete tasks

- Persistent data storage (Local Storage / Backend Database)

- Responsive UI (works on mobile and desktop)
